# Sorting-Visualizer
Demo -> https://mdfaiza007.github.io/Sorting-Visualizer
About the Project

Sorting Visualizer is a web-based project that visually demonstrates how different sorting algorithms work.
It helps users understand the internal working of algorithms by animating each step of the sorting process.
This project is built using HTML, CSS, JavaScript, and Data Structures & Algorithms (DSA) concepts.

#Features

Visual representation of sorting algorithms
Step-by-step animation of comparisons and swaps
Adjustable array size and speed
User-friendly and interactive UI
Helps beginners understand DSA concepts easily

#Algorithms Implemented

Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort

#Tech Stack
HTML – Structure of the web page
CSS – Styling and animations
JavaScript – Logic and DOM manipulation
DSA – Sorting algorithms implementation
